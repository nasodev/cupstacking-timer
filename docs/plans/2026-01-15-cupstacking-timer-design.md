# 컵쌓기 타이머 앱 설계

## 개요

스포츠 스태킹(컵쌓기) 시간 측정용 PWA 앱. 태블릿을 앞에 두고 화면 어디를 터치하든 타이머를 시작/종료할 수 있다.

## 요구사항

- 최초 진입 시 이름 입력
- 종목 선택 후 타이머 시작
- 각 종목별 최고 기록 저장
- 종목별 1등 표시
- 종합 순위: 가장 많은 종목에서 1등인 사람이 종합 1등

## 종목

**개인 종목:**
- 3-3-3
- 3-6-3
- 사이클

**단체 종목:**
- 더블
- 팀 3-6-3

## 화면 흐름

```
[시작 화면]
    │
    ▼
[이름 입력] ← 최초 1회 (로컬 저장, 이후 선수 목록에 추가됨)
    │
    ▼
[메인 화면] ─────────────────────────────────────┐
    │                                            │
    ├── 개인 종목 (3-3-3 / 3-6-3 / 사이클)         │
    │         │                                  │
    │         ▼                                  │
    │   [선수 선택] → [타이머] → [결과]            │
    │                                            │
    ├── 단체 종목 (더블 / 팀 3-6-3)               │
    │         │                                  │
    │         ▼                                  │
    │   [참가자 2명 이상 선택] → [타이머] → [결과]  │
    │                                            │
    ├── 순위 보기                                 │
    │     - 종목별 순위 (개인/단체 각각)           │
    │     - 종합 순위 (1등 개수 기준)              │
    │                                            │
    └── 선수 관리 (추가/삭제)                      │
─────────────────────────────────────────────────┘
```

## 데이터 구조

```typescript
// 선수 정보
interface Player {
  id: string;
  name: string;
  createdAt: Date;
}

// 기록 정보
interface Record {
  id: string;
  eventType: "3-3-3" | "3-6-3" | "cycle" | "doubles" | "team-3-6-3";
  playerIds: string[];  // 개인: 1명, 단체: 2명 이상
  time: number;         // 밀리초 단위 (예: 5320 = 5.320초)
  createdAt: Date;
}
```

**저장 위치:** 브라우저 LocalStorage
- `cupstacking_players` → 선수 목록
- `cupstacking_records` → 전체 기록

## 타이머 화면 UI

```
┌─────────────────────────────────────────┐
│                                         │
│            3-6-3                        │  ← 종목명
│           철수                          │  ← 선수명 (단체: "철수, 영희")
│                                         │
│                                         │
│                                         │
│          0.000                          │  ← 시간 (큰 글씨, 중앙)
│                                         │
│                                         │
│                                         │
│     화면 아무 곳이나 터치하세요            │  ← 안내 문구
│                                         │
└─────────────────────────────────────────┘
```

**상태 변화:**
1. 대기 중: "터치하면 시작" + 시간 0.000 (흰 배경)
2. 진행 중: 시간 증가 중 (초록 배경) + "터치하면 정지"
3. 완료: 최종 시간 표시 (노랑 배경) + "기록 저장됨" + [다시하기] [메인으로] 버튼

**핵심 포인트:**
- 전체 화면이 터치 영역 (버튼 찾을 필요 없음)
- 상태별 배경색으로 직관적 구분
- 시간은 소수점 3자리 (0.001초 단위)

## 기술 스택

- **프레임워크:** React + TypeScript
- **스타일링:** Tailwind CSS
- **빌드 도구:** Vite
- **PWA:** vite-plugin-pwa

## 프로젝트 구조

```
src/
├── pages/
│   ├── Home.tsx          # 메인 화면 (종목 선택)
│   ├── PlayerSelect.tsx  # 선수 선택
│   ├── Timer.tsx         # 타이머 화면
│   ├── Result.tsx        # 결과 화면
│   ├── Ranking.tsx       # 순위 보기
│   └── PlayerManage.tsx  # 선수 관리
├── components/
│   └── ... (공통 컴포넌트)
├── hooks/
│   ├── useTimer.ts       # 타이머 로직
│   └── useLocalStorage.ts # 로컬 저장
└── types/
    └── index.ts          # 타입 정의
```

## PWA 기능

- 홈 화면에 앱 아이콘 추가 가능
- 오프라인에서도 작동
- 전체 화면 모드 지원
