# 컵쌓기 타이머

스포츠 스태킹(컵쌓기) 경기용 타이머 PWA 앱입니다.

## 기능

### 지원 종목
- **개인 종목**: 3-3-3, 3-6-3, 사이클
- **단체 종목**: 더블, 팀 3-6-3

### 주요 기능
- 터치/클릭/스페이스바로 타이머 시작/정지
- 선수 등록 및 관리
- 종목별 기록 저장 및 순위표
- 오프라인 지원 (PWA)

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과물 미리보기
npm run preview
```

## 기술 스택

- React 19
- TypeScript
- Vite 7
- Tailwind CSS v4
- React Router v7
- PWA (vite-plugin-pwa)

## PWA 설치

모바일 또는 데스크톱 브라우저에서 "홈 화면에 추가" 기능을 사용하여 앱처럼 설치할 수 있습니다.

- **iOS**: Safari에서 공유 버튼 → "홈 화면에 추가"
- **Android**: Chrome 메뉴 → "앱 설치" 또는 "홈 화면에 추가"
- **Desktop**: 주소창 우측 설치 아이콘 클릭
